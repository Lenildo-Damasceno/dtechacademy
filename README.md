# 🎓 D Tech Academy - Plataforma de Cursos 
Site institucional , uma plataforma dedicada ao ensino de computação e tecnologia em Natal/RN.

## 📋 Sobre o Projeto

A D Tech oferece cursos na área de tecnologia, com foco em desenvolvimento web, programação, banco de dados, segurança da informação e DevOps.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e layout responsivo
- **JavaScript** - Interatividade e validações

## 📁 Estrutura do Projeto

```
inovatech/
├── index.html              # Página principal
├── src/
│   ├── styles/            # Arquivos CSS
│   ├── scripts/           # Arquivos JavaScript
│   │   ├── script.js      # Script principal
│   │   ├── navegacao.js   # Navegação e menu
│   │   ├── cursos.js      # Gerenciamento de cursos
│   │   ├── formulario.js  # Validação de formulários
│   │   └── secretaria.js  # Secretaria virtual
│   ├── components/        # Componentes reutilizáveis
│   └── utils/            # Funções auxiliares
├── assets/
│   ├── images/           # Imagens do site
│   ├── icons/            # Ícones
│   └── fonts/            # Fontes customizadas
└── pages/
    ├── cursos.html           # Página de cursos
    ├── localizacao.html      # Localização com mapa
    └── secretaria-virtual.html # Formulários de contato e matrícula
```

## 📝 Validações Implementadas

### Formulário de Contato
- Nome: mínimo 3 caracteres
- Email: formato válido
- Telefone: formato (84) 99999-9999
- Mensagem: 10-500 caracteres

### Formulário de Matrícula
- CPF: validação completa com algoritmo
- Telefone: validação de 10 ou 11 dígitos
- Todos os campos obrigatórios validados
